Projects

Generating Insights with Jarvis Microsoft

An applied AI project in collaboration with Evolutioned, focused on using Microsofts open-source JARVIS framework to generate meaningful insights from real-world project datasets. The goal is to build a reusable backend process that applies existing language models to break down project contexts and surface insights in a fraction of the time it would take a human. The project emphasizes backend development using Python, with no frontend work, and showcases the practical use of commoditised AI to augment decision-making.

View Project

Development of WheelEasy Blog Platform

This blog platform is a full-stack web application built with Next.js, Supabase, and PostgreSQL. It allows users to view published blog posts with rich content, categories, and author profiles. The platform supports dynamic routing, pagination, and secure content fetching via Supabase’s backend-as-a-service. Designed with scalability in mind, it integrates both server-side and client-side rendering and is styled using Vanilla CSS and Material UI to match existing frontend architecture. This project demonstrates my ability to adapt to pre-existing stacks while building responsive, data-driven components and maintaining clean, modular code.
